Régulation PID¶

https://fr.wikipedia.org/wiki/Automatique

https://fr.wikipedia.org/wiki/R%C3%A9gulateur_PID

Notons $e(t)$ l'erreur commise par le systme asservir, $w(t)$ la consigne et $y(t)$ la mesure de la grandeur rgle

On a : $e(t) = w(t) - y(t)$

$v(t)$ est la perturbation qui affecte le systme asservir

Asservissons ce systme l'aide d'un PID caractris par :

  • un coefficient proportionnel K
  • une constante de temps intgrale $T_i$
  • une constante de temps drive $T_d$

La rponse du rgulateur PID $u(t)$ est donne par la relation suivante : \begin{equation} u(t) = K ( e(t) + \frac{1}{T_i}\int_{0}^{t} e(t) dt + T_d \frac{de}{dt} ) \end{equation}

Dans le domaine de Laplace, cette relation devient : \begin{equation} U(p) = K(1 + \frac{1}{T_i p} + T_d p) E(p) \end{equation}

En effet, on montre par intgration par partie que :

  • le Laplacien de l'intgale d'une fonction f vaut $\frac{1}{p}$ fois le laplacien de f
  • le laplacien de la drive d'une fonction f vaut $p$ fois le laplacien de la fonction f, une constante prs

cf https://fr.wikipedia.org/wiki/Transformation_de_Laplace

Pour rappel, le laplacien d'une fonction f(t) est :

\begin{equation} F(p) = \int_{0}^{\infty} e^{-pt} f(t) dt \end{equation}

Au final, on peut crire :

\begin{equation} U(p) = K(\frac{1+ T_i p + T_i T_d p^2}{T_i p} ) E(p) \end{equation}

On obtient l'expression de la fonction de transfert du rgulateur PID, note $G_c$ :

\begin{equation} G_c = K(\frac{1+ T_i p + T_i T_d p^2}{T_i p} ) \end{equation}

Si l'on considre le systme dans son ensemble et que l'on note $G_a$ la fonction de transfert de la machine asservir, on a :

\begin{equation} Y(p) = G_a * G_c * (W(p) - Y(p)) \end{equation}

ou encore :

\begin{equation} Y(p) = \frac{G_a * G_c}{1+ G_a * G_c} * W(p) \end{equation}

On obtient l'expression de la fonction de transfert du systme complet :

\begin{equation} H_c(p) = \frac{G_a * G_c}{1+ G_a * G_c} \end{equation}

Réponse indicielle¶

La rponse indicielle d'un systme s'obtient sous la sollicitation d'un chelon, ie d'une fonction marche de Heavyside

La transforme de Laplace de la fonction h(t) de Heavyside est 1/p

\begin{equation} H(p) = \int_{0}^{\infty} e^{-pt} h(t) dt = \frac{-1}{p} \int_{0}^{\infty} -pe^{-pt} dt = \frac{-1}{p} [e^{-pt}]_{0}^{\infty} = \frac{1}{p} \end{equation}

Si $H_c(p)$ est la fonction de transfert du systme, le laplacien de la rponse indicielle est donc :

\begin{equation} Y(p) = \frac{H_c(p)}{p} \end{equation}

def F(p):
    return 4/(1+2*z*p/wn+(p/wn)**2)/p

# length to simulate in seconds
length=3
# step duration in seconds
step=0.05
# nb is number of steps to simulate  
nb=int(length//step)

# some numpy array to store the values
_target=4*np.ones(nb)
_lpz1=np.zeros(nb)
_lp1z=np.zeros(nb)
time=step
for i in range(0,nb):

    z=0.45
    _lpz1[i]=invertlaplace(F,time,method='talbot')

    z=1.12

    time+=step

On a utilis la fonction invertlaplace de la bibliothque mpmath mais la fonction de transfert du second ordre dont on est parti est le laplacien d'une fonction dont l'expression est connue.

\begin{equation} F(p)=\frac{w_n^2}{p^2+2zw_np+w_n^2} =\frac{1}{1+2\frac{z}{w_n}p+(\frac{p}{w_n})^2} \end{equation}

est le Laplacien de :

\begin{equation} f(t)=\frac{w_n}{\sqrt{1-z^2}}e^{-zw_nt}sin(w_nt\sqrt{1-z^2}) \end{equation}

et

\begin{equation} F(p)=\frac{w_n^2}{p(p^2+2zw_np+w_n^2)} =\frac{1}{p(1+2\frac{z}{w_n}p+(\frac{p}{w_n})^2)} \end{equation}

est le Laplacien de :

\begin{equation} f(t)=1-\frac{1}{\sqrt{1-z^2}}e^{-zw_nt}sin(w_nt\sqrt{1-z^2}+arcos(z)) \end{equation}
